Jolley Foxtrotters modern website
=================================

What this is:
- A modern, responsive static HTML/CSS/JS version of the mirrored Jolley Foxtrotters website.
- All real public content pages from the mirror were rebuilt:
  22 pages
- Original image folders were copied over so existing photos and links keep working.
- Added: mobile menu, responsive photo layouts, image lightbox, robots.txt, sitemap.xml, and 404.html.

Open this file first:
  index.html

Upload instructions:
1. Upload the CONTENTS of this folder to your web host / Cloudflare Pages project.
2. Keep the folders named horses, image, images, index, sold, and assets with the HTML files.
3. If you deploy somewhere other than https://www.mofoxtrot.com/jolley/, edit sitemap.xml and robots.txt to use the new domain.

Note about missing source pages:
HTTrack downloaded these as server 404 error pages, so they were not included as real content pages:
  - Kolbey Ballowe and Midnight Mandy, Missouri Fox Trotting Horse.html
  - Little Whiskey Girl and Kolbey, Missouri Foxtrotter Pony.html
  - Little Whiskey Girl and Kolbey.html

  - Super Mario, Missouri Fox Trotting Stallion.html

Footer note:
The old visitor counter depended on mofoxtrot.com's CGI counter. The modern footer preserves “Visitors since March 2003,” but the external CGI counter image was not reused because it will not work reliably on static hosting.
